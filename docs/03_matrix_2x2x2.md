# Матрица 2×2×2

Оси:

- **Активность**: активный / пассивный
- **Масштабируемость**: линейный / масштабируемый
- **Риск**: низкий / высокий

Получается 8 ячеек:

- A1 — активно, линейно, низкий риск
- A2 — активно, масштабируемо, низкий риск
- A3 — активно, линейно, высокий риск
- A4 — активно, масштабируемо, высокий риск
- P1 — пассивно, линейно, низкий риск
- P2 — пассивно, масштабируемо, низкий риск
- P3 — пассивно, линейно, высокий риск
- P4 — пассивно, масштабируемо, высокий риск

Каждая ячейка содержит краткое описание и типовые примеры.

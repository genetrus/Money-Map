# Формула из 3 частей

Каждый способ заработка описывается тремя вопросами:

1. **Что продаём (sell)** — время, результат, риск, доступ, внимание, капитал, собственность.
2. **Кому продаём (to_whom)** — одному клиенту, многим людям, рынку, государству, платформе.
3. **Как измеряется ценность (value_measure)** — ставка, цена, процент, рента, выплата, прирост.

Эта формула используется в таксономии и в классификаторе, чтобы связывать идеи
с ячейками матрицы и находить близкие механизмы.

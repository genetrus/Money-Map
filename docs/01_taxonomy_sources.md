# Таксономия механизмов (14)

Ниже список механизмов и ключевые теги (что продаём / кому / мера ценности):

1. **Зарплата** — время → платформа/клиент → ставка.
2. **Гонорар за результат** — результат → клиент/платформа → цена.
3. **Комиссия** — результат/риск → рынок/платформа → процент.
4. **Предпринимательская маржа** — доступ/результат/риск → рынок → цена.
5. **Рента/доступ** — доступ/собственность → рынок → рента.
6. **Проценты/дивиденды** — капитал → рынок → выплата.
7. **Прирост стоимости** — капитал → рынок → прирост.
8. **Интеллектуальная собственность** — собственность/результат → рынок → цена/рента.
9. **Внимание/аудитория** — внимание → платформа → выплата/процент.
10. **Данные/вычисления** — доступ/результат → рынок/платформа → цена.
11. **Конкурсы/гранты** — результат → государство → выплата.
12. **Трансферты/страховые** — государство → выплата.
13. **Подарки/наследство** — отношения → выплата.
14. **Экономия как доход** — эффект на бюджет → устойчивость.

Эта карта помогает быстро определить, какой механизм у текущей идеи и в каких
ячейках матрицы он обычно лежит.

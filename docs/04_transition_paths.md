# Типовые маршруты переходов

Маршруты показывают типичный порядок развития:

- **Здоровый коридор**: A1 → A2 → P1 → P2
- **Быстрый кэш**: A1 → A3 → A2
- **Предпринимательский рост**: A1 → A2 → A4 → P1 → P2
- **Инвесторский (консервативный)**: A1 → P1
- **Платформа-лотерея**: A1 → P4 → P2

Идея в том, чтобы осознанно выбирать путь и понимать компромиссы по риску и масштабу.

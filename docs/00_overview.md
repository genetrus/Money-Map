# Обзор

Money Map — это модель путей заработка, где каждую идею можно описать через:

1. **Что продаём** — время, результат, риск, доступ, внимание, капитал, собственность.
2. **Кому продаём** — одному клиенту, многим людям, рынку, государству, платформе.
3. **Как измеряется ценность** — ставка, цена, процент, рента, выплата, прирост стоимости.

Вторая часть — матрица 2×2×2, где оси:

- Активность: активный / пассивный
- Масштабируемость: линейный / масштабируемый
- Риск: низкий / высокий

Матрица создаёт 8 ячеек (A1..P4). Мосты и маршруты показывают, как люди обычно
перемещаются между ячейками.

## Как пользоваться

- Проверьте данные: `money-map validate`
- Посмотрите матрицу и механизмы: `money-map cells`, `money-map taxonomy`
- Постройте экспорт: `money-map export all`

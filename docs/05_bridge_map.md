# Карта мостов

Мосты описывают механизмы перехода между ячейками:

- A1 → A2: стандартизация, упаковка, ретейнеры, делегирование, автоматизация, канал продаж, сегментация.
- A2 → P2: артефакт, продуктивизация, воронка.
- A1 → P1 / A2 → P1: авто-накопление.
- A1 → A3: прыжок в комиссию/успех.
- A2 → A4: агрессивное масштабирование.
- A1 → P4: ставка на вирусность.
- P1 → P3: повышение риска на капитале.

## Опасные мосты

Мосты, связанные с ростом риска или зависимостью от удачи:

- A1 → A3
- A2 → A4
- A1 → P4
- P1 → P3

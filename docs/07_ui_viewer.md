# UI Viewer (Streamlit)

## Назначение

Графический интерфейс Money Map позволяет работать с матрицей 2×2×2, мостами,
маршрутами и таксономией в виде приложением-окна, а также выполнять поиск и
классификацию прямо из браузера.

## Запуск

1. Установите дополнительные зависимости:

```bash
pip install -e ".[ui]"
```

2. Запустите UI через CLI:

```bash
money-map ui
```

3. Альтернативный запуск напрямую через Streamlit:

```bash
streamlit run -m money_map.ui.app
```

## Возможности

- Матрица 2×2×2 с кликабельными карточками ячеек.
- Списки мостов и маршрутов с фильтрами и деталями.
- Способы получения денег: карта и справочник с деталями, связанными мостами и маршрутами.
- Граф "14 способов" внутри раздела "Способы получения денег" с фиксированной звездой и тегами.
- Глобальный полнотекстовый поиск по ячейкам, мостам, маршрутам и таксономии.
- Классификатор: структурированный (теги) и текстовый режимы.
- Интерактивный граф переходов с экспортом DOT.

## Способы получения денег

Раздел объединяет карту и справочник. На карте показана звезда таксономии: центр "Способы получения денег",
14 механизмов по окружности и (опционально) кольцо тегов sell/to_whom/value. Выбор механизма на карте
автоматически открывает справочник и показывает детали (связанные ячейки, мосты и маршруты).

## Данные

UI использует те же YAML-данные, что и CLI. Папку данных можно переопределить
через переменную окружения `MONEY_MAP_DATA_DIR`.
